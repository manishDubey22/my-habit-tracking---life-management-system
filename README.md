# my-habit-tracking---life-management-system

A smart habit tracking app that helps users plan daily routines, track completion in a calendar-style grid, and visualize progress with insights. It transforms habits into measurable data to improve consistency, productivity, and self-improvement.

## Tech stack

- React + TypeScript
- Vite
- Zustand
- MUI
- Recharts

## Getting started

```bash
npm install
npm run dev
```

## Available scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
